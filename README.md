# AI Explanation Generator

## Problem
Users find it difficult to understand technical topics at different levels.

## Solution
Built an AI-powered application that explains topics based on user-selected level (Beginner, Intermediate, Expert).

## Tech Used
- Python
- Streamlit
- Groq API

## Features
- Topic input
- Level selection
- Dynamic explanation based on difficulty
- Clean UI

## How to Run
pip install -r requirements.txt
python -m streamlit run app.py
